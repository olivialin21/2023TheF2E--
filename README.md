### 執行方式
1. 下載 **node.js**
2. 下載 **yarn**（開啟命令提示字元執行 ```npm install -g yarn``` ）
3. 在命令提示字元中移動到想下載檔案的位置，輸入 ```git clone https://github.com/olivialin21/2023TheF2E-Misson1```
4. ```cd 2023TheF2E-Misson1```
5. ```yarn start```
6. 成功開啟專案
